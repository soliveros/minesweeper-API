## Description
REST API implementation for Mine Sweeper game created with Nest JS Framework.  
[Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ npm install
```

## Run with docker

```bash
$ docker-compose up -d
```

## Running the app without docker
Warning: You need to have a local instance of Mongo DB  (It is included in docker-compose file)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
